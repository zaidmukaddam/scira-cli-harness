## Benchmark run logs

The folder **`benchmark-runs/`** holds **committed** Markdown (or JSON) exports from sweeps we cite in public posts. Keep them small and redacted. **Personal or experimental runs** go in repo-root **`benchmark-results/`**, which is gitignored.
